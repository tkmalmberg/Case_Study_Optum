Selecting Elements By ID

An ID should be unique within a page. However, if more than one element with the specified ID exists, the getElementById() method returns the first element in the source code.

getElementsByName() will also return elements that have an id attribute with the specified value. For compatibility, you should be careful not to use the same string as both a name and an ID.

The Document method querySelector() returns the first Element within the document that matches the specified selector, or group of selectors, this must be valid CSS syntax, or a SyntaxError exception will occur. Elements can be described by ID, tag name, or class


Root Nodes

  • document
  • document.documentElement
  • document.head
  • document.body

Parent Nodes

  • Node.parentNode:The parent of an element is an Element node, a Document node, or a DocumentFragment node
  • Node.parentElement:This is always a DOM Element object, or null
